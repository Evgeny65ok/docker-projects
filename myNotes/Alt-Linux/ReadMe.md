# Отчет: Работа с российским дистрибутивом Alt Linux в Docker

## 1. Запуск системы
Был загружен и запущен образ ветки `sisyphus` (нестабильная ветка для разработчиков):
`docker run -ti --rm --name alt alt:sisyphus /bin/bash`

## 2. Установка ПО
Для демонстрации работы менеджера пакетов `apt-get` была установлена утилита `fastfetch`.

## 3. Вывод
Контейнер Alt Linux успешно запущен. Взаимодействие с репозиториями Sisyphus через `apt-get` прошло без ошибок, утилита fastfetch корректно определила параметры системы.
